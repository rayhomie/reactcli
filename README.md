# @rayhomie/reactcli

### install

```bash
npm install @rayhomie/reactcli -g
```
### check

```bash 
# View version
reactcli -V
# or
reactcli --version

# for hlep
reactcli --help
```
### use

Open the terminal and enter the following to create the project
```bash
reactcli create your_project_name
```

### start

```bash
cd ./your_project_name
yarn ios 
# or
yarn android
```
